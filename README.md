#### Penjelasan kode 

1. Buat sebuah dictionary awal dengan nama data_mahasiswa untuk menyimpan data mahasiswa.
2. Gunakan fungsi def untuk mendefinisikan beberapa fungsi operasi data.
3. Definisikan fungsi look_data untuk menampilkan seluruh isi data mahasiswa dari dictionary.
4. Definisikan fungsi plus_data untuk menambahkan data mahasiswa baru ke dalam dictionary data_mahasiswa.
5. Definisikan fungsi change_data untuk mengubah data mahasiswa yang ada dalam dictionary berdasarkan NIM (Nomor Induk Mahasiswa).
6. Definisikan fungsi find_data untuk mencari dan menampilkan data mahasiswa tertentu dari dictionary menggunakan NIM.
7. Definisikan fungsi delete_data untuk menghapus data mahasiswa dari dictionary berdasarkan NIM.
8. Gunakan loop while True untuk menjalankan program secara berulang, dengan menu pilihan untuk pengguna.
9. Jika pengguna memilih opsi 'k' atau 'keluar', maka hentikan loop dan akhiri program.
